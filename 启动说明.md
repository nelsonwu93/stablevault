# 基金智能监控系统 - 启动说明

## 第一步：安装依赖（只需一次）

打开终端，进入本文件夹，执行：

```bash
pip install -r requirements.txt
```

## 第二步：启动系统

```bash
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501

## 系统功能说明

### 📊 持仓总览
- 所有11只基金实时状态
- 板块配置对比图（当前 vs 目标30万配置）
- 每只基金的止盈止损进度条

### 🌍 宏观监控
- 全球关键指标（纳斯达克、英伟达、汇率、美债）
- 美联储立场自动判断
- 北向资金近5日净流入
- 宏观指标→持仓基金影响对照表

### 💡 买卖建议（核心功能）
点击「运行 Checklist Agent」后，系统会输出：

**① 触发事件** - 什么信号引发了这次分析

**② 传导逻辑链路** - 逐步显示宏观因素如何影响这只基金的推理过程
  例如：美联储降息 → 美元走弱 → 全球风险偏好上升 → 科创板受益

**③ 数据佐证** - 支撑结论的具体数据

**④ Checklist Agent 8项核查** - 每项都显示：
  - 检查结论（通过/失败）
  - 具体数值和原因
  - 权重占比
  - 一票否决项特别标注

### 📁 决策记录
- 自动保存所有历史分析
- 可填写实际操作用于复盘
- 系统建议 vs 你的操作 vs 最终结果对比

## 常见问题

**Q: 数据获取失败怎么办？**
A: 系统直接调用东方财富公开API，需要稳定的网络连接。遇到失败可点击「刷新所有数据」重试。

**Q: 基金代码错误显示不了数据？**
A: 按上表说明在 config.py 中更新代码，然后重启 streamlit。

**Q: 建议是否一定要执行？**
A: 本系统仅供辅助参考，不构成投资建议。所有建议都展示了完整推理，请结合自身判断决策。
